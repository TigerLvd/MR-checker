Необходимо написать приложение по проверке изменений в MR из GitLab.

Планируется, что приложение будет вызываться в piplane, который отработывает при создании MR.
В приложении нужно запросить данные из GitLab (изменения по MR) и вызывать локальную LLM для проверки. После чего вернуть ответ в GitLab.

Технологии: приложение должно быть написано на Spring Boot
